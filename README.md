# How To Run?


1) Composer: `composer install`


2) Environment file: `cp .env.example .env`


3) Docker up:
   `APP_PORT=9090 ./vendor/bin/sail up -d
   `


4) Migrate command:
   `./vendor/bin/sail artisan migrate
   `


5) Seed command:
   `./vendor/bin/sail artisan db:seed
   `
